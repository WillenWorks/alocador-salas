# Configuração

## Variáveis de ambiente (Supabase)
Crie um arquivo `.env` na raiz do projeto (`website/alocador-salas/.env`) com:

```bash
VITE_SUPABASE_URL=
VITE_SUPABASE_ANON_KEY=
```

- Se essas variáveis não estiverem preenchidas, o app entra em **modo demo** (dados no navegador) apenas para testar a UI.
